## 用户管理

### 角色与权限

| 角色 | 权限 | 说明 |
|------|------|------|
| **所有者** | 完全访问 | 系统级管理 |
| **管理员** | 管理操作 | 用户管理、内容管理 |
| **编辑者** | 内容编辑 | 创建、编辑、发布文章 |
| **投稿者** | 文章创建 | 创建文章（需审核） |
| **成员** | 浏览、评论 | 普通用户 |

## AI 配置

### 模型路由

按任务配置使用的 AI 模型：

| 任务 | 推荐模型 | 备用模型 |
|------|---------|---------|
| 通用聊天 | GPT-4o / Gemini | DeepSeek |
| 中文优化 | Qwen / GLM | GPT-4o |
| 成本优先 | DeepSeek V3 | Qwen |
| 图像生成 | DALL-E | Tongyi Wanxiang |
| 语音对话 | Gemini Realtime | Qwen Voice |
