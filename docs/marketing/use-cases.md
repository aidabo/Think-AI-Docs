# ユースケース

## メディア・新聞業界

### 課題
新聞購読者の減少、広告収入の低下、若年層のニュース離れ。

### ソリューション

```mermaid
flowchart TB
    subgraph Newsroom["AI ニュースルーム"]
        WRITER[AI 記者アシスタント<br/>記事下書き / 要約 / 見出し生成]
        VOICE[AI 音声配信<br/>記事読み上げ / ポッドキャスト]
        PERSONALIZE[パーソナライズド<br/>ニュース配信]
        COMMUNITY[読者コミュニティ<br/>AI 管理コメント]
    end

    subgraph Results["導入効果"]
        R1[⚡ 記事作成時間 60% 削減]
        R2[📱 新しい読者層の獲得]
        R3[💬 コメント欄の質向上]
        R4[🌏 多言語配信が容易に]
    end

    Newsroom --> Results
```

**主要機能:**
- AI 記者アシスタント（下書き作成、見出し、ファクトチェック）
- パーソナライズドニュース配信
- AI 音声読み上げ（通勤・家事中でもニュースを「聴く」）
- AI コメントモデレーション
- Page Builder で速報ページをノーコード作成

## 不動産業界

### 課題
物件管理の非効率、顧客対応の属人化、海外投資家への情報発信不足。

### ソリューション

```mermaid
flowchart TB
    subgraph RE["不動産 DX"]
        AGENT[AI 物件コンシェルジュ<br/>24/365 自動応答]
        SEARCH[ハイブリッド検索<br/>AI 曖昧 + 条件フィルター]
        PAGE[Page Builder<br/>物件ページ自動作成]
        TRANS[AI 翻訳<br/>日・中・英自動翻訳]
        MIKU[AI マイソク<br/>自動生成]
        NOTIFY[スマート通知<br/>内見リマインド / 値下げ通知]
    end

    subgraph Effects["導入効果"]
        E1[📞 顧客応答率 3 倍]
        E2[⏱ ページ作成時間 90% 削減]
        E3[🌏 海外からの問合せ増加]
        E4[📋 マイソク作成が瞬時に]
    end

    RE --> Effects
```

**主要機能:**
- AI 物件コンシェルジュ（24 時間対応、多言語）
- 自然言語 + 条件のハイブリッド検索
- Page Builder 物件テンプレート
- AI 自動翻訳（日本語 ⇄ 中国語）

## 教育業界

### 課題
オンライン教育の質向上、個別指導の自動化、コミュニティ形成。

### ソリューション

| 機能 | 適用例 |
|------|--------|
| **AI チュータリング** | 学生の質問に 24 時間対応、個別学習プラン |
| **講座ページビルダー** | ノーコードで講座ページを作成・公開 |
| **コミュニティ機能** | クラス別グループ、ディスカッション管理 |
| **メディア処理** | 講義動画の自動文字起こし・翻訳 |
