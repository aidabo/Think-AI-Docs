# Think-AI 機能一覧

## 機能カテゴリ

```mermaid
graph TB
    subgraph Platform["プラットフォーム基盤"]
        CMS[Ghost CMS 互換<br/>200+ API エンドポイント]
        CUSTOM[カスタム Social API<br/>30+ /social/* エンドポイント]
        DOCKER[Docker コンテナ<br/>一発デプロイ]
    end

    subgraph AI["AI 機能"]
        CHAT[マルチモデルチャット<br/>GPT-4o / Gemini / DeepSeek / Qwen / GLM]
        VOICE[リアルタイム音声対話<br/>Gemini / OpenAI / Qwen Voice]
        IMAGE[AI 画像生成<br/>DALL-E / Wanxiang / Stable Diffusion]
        MEDIA[メディア自動処理<br/>文字起こし / 翻訳 / サムネイル]
        AGENT[AI エージェント<br/>画像 / 検索 / リマインダー / 音声]
    end

    subgraph SNS["SNS 機能"]
        GROUP[グループ管理<br/>オーナー / 管理者 / メンバー権限]
        COMMENT[拡張コメント<br/>いいね / 返信 / レポート / モデレーション]
        GRAPH[ソーシャルグラフ<br/>フォロー / ブックマーク / お気に入り]
        GALLERY[ギャラリー<br/>S3 署名付きアップロード]
    end

    subgraph Builder["Page Builder"]
        PB_DRAG[ドラッグ＆ドロップ<br/>ページ編集]
        PB_DATA[データバインディング<br/>API 連動]
        PB_TEMPLATE[テンプレート管理<br/>再利用可能]
        PB_PUBLIC[公開ページ<br/>外部共有]
    end

    subgraph Operation["運用機能"]
        MONITOR[Prometheus 監視]
        ANALYTICS[Tinybird 分析]
        AUDIT[操作監査ログ]
        ACTIVITY[ユーザーアクティビティ]
    end

    Platform --> AI
    Platform --> SNS
    Platform --> Builder
    Platform --> Operation
```
