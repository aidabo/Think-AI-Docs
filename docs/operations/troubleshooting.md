# トラブルシューティング

## よくある問題と解決策

### デプロイ・起動関連

| 問題 | 原因 | 解決策 |
|------|------|--------|
| `docker compose up` が失敗する | ポート競合 | `docker ps` で使用中ポートを確認、競合サービスを停止 |
| MySQL 接続エラー | コンテナ起動順 | `docker compose restart ghost` で再起動 |
| 403 Forbidden | API キーが未設定 | `.env` で `API_KEY` を確認 |
| 502 Bad Gateway | コンテナ未起動 | `docker compose ps` で全コンテナの状態を確認 |

### AI 機能関連

| 問題 | 原因 | 解決策 |
|------|------|--------|
| AI 応答がない | API キーが無効 | 管理画面で AI 設定を確認 |
| 音声が聞こえない | マイク権限 | ブラウザのマイク権限設定を確認 |
| 画像が生成されない | プロバイダ制限 | DALL-E のレート制限を確認 |
| ストリーミングが途切れる | ネットワーク | WebSocket 接続を確認 |

### パフォーマンス関連

| 問題 | 原因 | 解決策 |
|------|------|--------|
| ページ読み込みが遅い | キャッシュ不足 | SWR キャッシュ設定を確認 |
| AI 応答が遅い | プロバイダ遅延 | 管理画面でモデルを切り替え |
| 画像アップロードが遅い | S3 リージョン | サーバーと同じリージョンを使用 |

### データ関連

| 問題 | 原因 | 解決策 |
|------|------|--------|
| データが消えた | バックアップ未取得 | 日次バックアップを設定 |
| DB 接続エラー | MySQL 停止 | `docker compose logs mysql` で確認 |
| S3 画像が表示されない | CORS 設定 | S3 バケットの CORS 設定を確認 |

## ログの確認方法

```bash
# バックエンドログ
docker compose logs ghost

# フロントエンドログ
docker compose logs nextjs

# DB ログ
docker compose logs mysql

# メディアワーカーログ
docker compose logs media-worker
```

## システムヘルスチェック

```bash
# 全コンテナの状態確認
docker compose ps

# リソース使用状況
docker stats

# API ヘルスチェック
curl http://localhost:2368/ghost/api/admin/health

# ディスク使用量
df -h
```
